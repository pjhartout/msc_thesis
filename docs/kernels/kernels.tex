\begin{frame}[fragile]{W-L kernel}
  \begin{itemize}
  \item Computing $\phi(G)$ needs to be done explicitely and can be done
    independently prior to computing $K_{WL}=\phi(G)^T\phi(G')$
  \item How to compute $\phi(G)$? \texttt{networkx} has a function called
\href{https://networkx.org/documentation/stable/reference/algorithms/generated/networkx.algorithms.graph_hashing.weisfeiler_lehman_graph_hash.html}{\texttt{weisfeiler\_lehman\_graph\_hash}}
but it does not return the hash counts at each iteration.
  \end{itemize}
\end{frame}
